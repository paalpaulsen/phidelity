Phidelity by Interfazer

Phidelity is a responsive design framework made for precision, harmony and flexibility. The system combines Phi "the golden ratio" (1:1.618) with the modularity of Lego.

Below is a demo with some container examples and how they are defined using container queries and css grid - using grid areas. Try expanding the browser on a > 3200px wide screen to see how it responds as containers are nested inside wrappers. No extra code for the containers needed. Containers in wrappers create another layer of layout possibility, to cater for increasingly wider screens.

Created by Pål Eirik Paulsen
